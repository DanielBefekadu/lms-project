Learning Management System (LMS)

Overview
This project is a Learning Management System (LMS) built using PHP, MySQL, HTML, and CSS as part of the RITA Africa Web Development Training.
It allows students to register, log in securely, and access their dashboard within a simple and modern user interface.

Features

        -User Registration & Login Authentication

        -Session-based Security

        -Modern UI Design using HTML & CSS

        -MySQL Database Integration

        -PHP Backend Logic for data handling

        -Organized Folder Structure (config/, assets/, views/, etc.)

Technologies Used:

Frontend	:  HTML5, CSS3
Backend	  :  PHP 8+
Database	:  MySQL
Tools     : Git , GitHub , VS Code
